CSS Transitions & Animations

Transitions

Transitions and animations are similar, but are not the same.

Transitions are used for transitioning properties from one state to another.

The Syntax

transition-property: all | background | color | height | width ...
transition-duration: 100ms | .1s 
transition-timing-function: ease | linear
